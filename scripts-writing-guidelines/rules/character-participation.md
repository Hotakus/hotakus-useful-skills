# 角色参与规则

## 1. 核心原则

> 当角色 A 在场时，角色 B 的情感转折必须有角色 A 的参与——哪怕只是一个点头、一个动作、一行对话。禁止角色独自完成"观察→困惑→确认"的完整情感弧线，而在场的他人像空气一样沉默。

## 2. 反例

```
> 角色B：「你摸过它。我在梦里看到的。」
角色B转头看角色A。笑了笑。
> 角色B：「不是梦——是你真的来过。」
```

**问题**：角色B 自问自答完成了整个"是梦→不是梦"的认知转折。角色A 站在旁边，零参与——情感转折成了独角戏。

## 3. 正例

```
> 角色B：「你摸过它。我在梦里看到的。」
角色B转头看角色A。
> 角色B：「……是真的吗？」
（角色A 点头。）
角色B 笑了笑。
> 角色B：「原来是真的。你真的来过。」
```

**改进**：角色B 把判定的权力交给角色A。角色A 一个点头给出了答案。角色B 的笑和"原来是真的"才有了落点——情感转折从"独角戏"变成了"互动"。

## 4. 修复模板

对于每个"沉默旁观"场景，插入 1-2 拍在场他人的反应：

| 类型 | 示例 |
|------|------|
| 非语言反应 | 角色A 的手停在半空。他没有说话——但他在听。 |
| 最小动作 | 角色A 往前走了一步。然后停住——把空间留给对方。 |
| 一行对话 | 角色A："……嗯。" |
| 交互触发 | 角色B 看向角色A。角色A 点头。角色B 笑了笑——然后继续。 |

## 5. 常见违规模式

| 模式 | 表现 | 修复方向 |
|------|------|---------|
| 自问自答 | 角色提出疑问，然后自己回答，不给在场他人反应空间 | 把疑问抛给在场的人——哪怕只是等待一个眼神 |
| 不等回答 | 角色表白/交心后，"没等回答"就走 | 删除"没等回答"——给对方一个拍子的反应时间 |
| 全场沉默 | 角色完成一整段脆弱倾诉，在场的多人没有任何反应 | 至少一个人有动作——靠近、低头、转开视线 |
| 替代式支持 | 角色A 本该提供的情感支持，被角色C 替代完成 | 让角色A 与角色C 共享这个时刻，而非被替代 |

### 5.1 动机可见性

当角色在情感场景中**单独提及另一角色**时（"尤其是XX"、"别告诉XX"、"和XX不一样"），观众必须能理解为什么。动机不能只存在于作者的脑子里。

**反例**：
```
> 埃莉诺：「……别跟别人说。尤其是莉赛特。」
```
玩家不知道埃莉诺为什么特别点名莉赛特。

**正例**：
```
> 埃莉诺：「……别跟别人说。尤其是莉赛特——她连家都没了。我找到了人。还在怕。算什么。」
```
埃莉诺用自己的声音说出了原因：和莉赛特的绝对失去相比，她的恐惧令她羞耻。

> 原则：角色自己说出动机，不用旁白翻译。三句碎片比一段旁白更有力。

### 5.2 反应必须有意义

插入的角色反应必须**传达信息给观众**。不能只是为了"填充沉默"而添加无意义的物理细节。

**反例**：
```
火山灰落在他肩膀上。他没有拍掉。
```
观众无法理解这个细节要表达什么——他在专心听？他不在乎灰？他在发呆？没有任何信息。

**正例**：
```
他等她说完。然后才伸手——把箭袋歪了的带子拨正。
```
动作传达了具体信息：他在听，也在观察，而且等她说完才动——尊重。

> 检验标准：观众读到这个动作时，能说出它表达了什么。如果不能——删掉。

### 5.3 避免否定句式重复

"主角没有说话"、"主角没有回答"、"主角没有回头"——这些否定句式用一两次可以，泛滥就成了作者的偷懒。优先用**肯定句**（他做了什么），而非否定句（他没做什么）。

**反例**：
```
主角没有回答。但他没有转开视线——他看着她。然后点了一下头。
```
叠了两个"没有"。

**正例**：
```
他看着她。然后点了一下头。
```
删掉否定前缀，直接说动作。

> 自检：搜全文 `主角没有`，如果超过 3 处，逐处替换为肯定句。

### 5.4 表述歧义检查

台词的常见联想义必须与角色设定/战斗定位一致。当字面意思引发错误联想时，换表述。

检验标准：读者的第一反应是否符合设定意图。

| 反例 | 问题 | 正例 |
|------|------|------|
| "在你战斗的时候——站在你身后。" | "站在身后"字面联想为躲后面，与战前配置/羁绊辅助的主动定位冲突 | "你在前面的时候——我的力量会辅助你。" |
| "我跟你一起扛。"（非体力场景） | "扛"字有体力活联想，在情感/魔法/远程辅助等场景不适用 | "我跟你一起分担。" |

> "分担"强调共同面对，不预设场景类型。"扛"仅适用于体力劳动或战斗前锋。

### 5.5 防全知

虽然上下文中拥有角色和世界的全部信息，但在写作过程中，必须站在角色的角度，思考这个角色**现在知道些什么**。坚决避免角色说出他不知道的信息。

检验标准：向前追溯，能找到角色获知该信息的镜头——看见、听见、触到、被告诉。如果追溯失败，就是作者在替角色说话。

| 反例 | 缺失 | 正例 |
|------|------|------|
| 露西娅说"她体内有神之遗赠" | 露西娅从未见过莉奥拉，无法知道遗赠存在 | 露西娅看到光 → 认出是自然神的力量（圣女身份）→ 说出判断 |
| 莉奥拉说"该我帮你了"（刚看到光，未见效果） | 光≠有用。角色未见证据就下了结论 | 光漫到剑上 → 剑发光 → 莉奥拉看到 → "或许能帮上你了" |
| 光自动飘向主角 | 为什么飘？读者和角色都不知道 | 莉奥拉主动伸手碰主角 → 光自然过渡（无意，但因果可见） |

> 原则：角色的认知是感官驱动的——看见、听见、触到、被告诉。如果认知来源不在剧本中，它就是作者在替角色说话。

## 6. 注意

- 沉默本身可以是一种有力的交流方式。关键是沉默是**选择**还是**缺席**——如果剧本描述了角色A 的微动作，沉默就是选择；如果完全没有提及角色A，沉默就是缺席。
- 群戏场景的自然沉默可以保留——围坐篝火、全体宣言等场景中，沉默可能是共享的、舒适的。
- 角色设定可以解释沉默——即使角色A 是寡言的人，也需要一个动作或眼神来证明"他在"。

## 7. 自检要求

输出前请自检，如出现违反以上规则的输出，重新输出。
