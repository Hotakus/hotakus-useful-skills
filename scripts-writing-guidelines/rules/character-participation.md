# 角色参与规则

## 1. 核心原则

> 当角色 A 在场时，角色 B 的情感转折必须有角色 A 的参与——哪怕只是一个点头、一个动作、一行对话。禁止角色独自完成"观察→困惑→确认"的完整情感弧线，而在场的他人像空气一样沉默。

## 2. 反例

```
> 角色B：「你摸过它。我在梦里看到的。」
角色B转头看角色A。笑了笑。
> 角色B：「不是梦——是你真的来过。」
```

**问题**：角色B 自问自答完成了整个"是梦→不是梦"的认知转折。角色A 站在旁边，零参与——情感转折成了独角戏。

## 3. 正例

```
> 角色B：「你摸过它。我在梦里看到的。」
角色B转头看角色A。
> 角色B：「……是真的吗？」
（角色A 点头。）
角色B 笑了笑。
> 角色B：「原来是真的。你真的来过。」
```

**改进**：角色B 把判定的权力交给角色A。角色A 一个点头给出了答案。角色B 的笑和"原来是真的"才有了落点——情感转折从"独角戏"变成了"互动"。

## 4. 修复模板

对于每个"沉默旁观"场景，插入 1-2 拍在场他人的反应：

| 类型 | 示例 |
|------|------|
| 非语言反应 | 角色A 的手停在半空。他没有说话——但他在听。 |
| 最小动作 | 角色A 往前走了一步。然后停住——把空间留给对方。 |
| 一行对话 | 角色A："……嗯。" |
| 交互触发 | 角色B 看向角色A。角色A 点头。角色B 笑了笑——然后继续。 |

## 5. 常见违规模式

| 模式 | 表现 | 修复方向 |
|------|------|---------|
| 自问自答 | 角色提出疑问，然后自己回答，不给在场他人反应空间 | 把疑问抛给在场的人——哪怕只是等待一个眼神 |
| 不等回答 | 角色表白/交心后，"没等回答"就走 | 删除"没等回答"——给对方一个拍子的反应时间 |
| 全场沉默 | 角色完成一整段脆弱倾诉，在场的多人没有任何反应 | 至少一个人有动作——靠近、低头、转开视线 |
| 替代式支持 | 角色A 本该提供的情感支持，被角色C 替代完成 | 让角色A 与角色C 共享这个时刻，而非被替代 |

## 6. 注意

- 沉默本身可以是一种有力的交流方式。关键是沉默是**选择**还是**缺席**——如果剧本描述了角色A 的微动作，沉默就是选择；如果完全没有提及角色A，沉默就是缺席。
- 群戏场景的自然沉默可以保留——围坐篝火、全体宣言等场景中，沉默可能是共享的、舒适的。
- 角色设定可以解释沉默——即使角色A 是寡言的人，也需要一个动作或眼神来证明"他在"。
