# 去八股 / De-formula

写作时避免八股文式表达。适用于中文创意写作——对话、叙述、描写。

## 1. 禁止否定前置

严禁以下结构，它们用否定来制造虚假的修辞张力：

| 句式 | 示例 |
|------|------|
| 不是A，而是B | 语气不是疑问，是陈述 |
| 不是A，是B | 那不是害怕，是紧张 |
| 不是A，更像B | 不像生气，更像委屈 |
| A？不，B | 害怕？不，是兴奋 |
| 与其说A，不如说B | 与其说生气，不如说无奈 |
| 没说...也没说 | 没说信，也没说不信 |

替代方案：
- 直接说事实——「她用的是陈述句。」
- 用"但"连接肯定句——「她看起来冷淡，但藏着一丝关切。」
- 拆独立短句——「她没说话。只是静静看着。」

## 2. 禁止抽象描写

| 禁止 | 原因 |
|------|------|
| 语气描写 | "语气里带着一丝..."——读者无法体会 |
| 目光/眼神 | "目光落在光上，停了一会儿"——无意义废话 |
| 冗余动作堆叠 | "拧开又拧上，手指摩挲"——无用细节 |

替代方案：
- 具体动作替代抽象——「她的手一直攥着衣角」
- 对话替代内心——说出来，而不是"他在想"
- 砍掉多余的形容词和副词

## 3. 自检要求

输出前请自检，如出现违反以上规则的输出，重新输出。
